IQ-Leaf ‚Äì Smart Plant Care System

IQ-Leaf is an end-to-end IoT solution for plant care, integrating sensors, actuators, custom hardware, and both local and remote interfaces.

  • Real-time soil moisture sensing via capacitive sensor on ESP32 ADC (more durable than resistive probes).
  • 5V water pump controlled by relay module with guardrails to avoid over-watering and pump burnout.
  • Custom PCB (Eagle CAD) combining ESP32-S3, power regulation, relay driver, and modular connectors.
  • Responsive web dashboard served from the ESP32 using an async web server (HTML/CSS/JS).
  • ESP32-CAM video stream for live visual checks, integrated into the web UI.
  • TFT LCD over SPI showing moisture level, status, and Wi-Fi details.
  • Core logic written in C/C++ (Arduino framework) and tracked with Git.

Tech Highlights: ESP32-S3, ESP32-CAM, C/C++, Arduino, Eagle CAD, async web server, IoT design, embedded debugging.
