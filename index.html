IQ-Leaf System

IQ-Leaf is an end-to-end IoT solution for plant care, integrating sensors, actuators, custom hardware, and both local and remote interfaces. It showcases embedded programming, PCB design, and web development.

Key Features & Implementation Details:

  • Real-time soil moisture sensing using a capacitive sensor connected to the ESP32’s ADC.
  • Automated watering via a 5V pump and relay, with safeguards to prevent over-watering.
  • Custom PCB designed in Eagle CAD integrating ESP32-S3, power regulation, relay driver, and sensor/connectors.
  • Responsive web dashboard served from the ESP32 using an async web server (HTML, CSS, JavaScript) over Wi-Fi.
  • ESP32-CAM video stream for visual checks, accessible through the same interface.
  • TFT LCD status display (SPI) for moisture levels, system state, and connection info.
  • Core logic written in C/C++ (Arduino framework) with libraries for Wi-Fi, web server, display, and camera.
  • Version control with Git throughout development.

Technologies Used: ESP32-S3, ESP32-CAM, C/C++, Arduino Framework, HTML, CSS, JavaScript, Eagle CAD, capacitive moisture sensor, TFT LCD, relay module, Wi-Fi, async web server, Git.
